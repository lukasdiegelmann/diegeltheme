# diegeltheme

A pitch-black Omarchy theme for Hyprland that delivers the deep, inky blacks your OLED display craves.

## Installation

```bash
omarchy-theme-install https://github.com/lukasdiegelmann/diegeltheme.git
```

## Usage

Once installed, press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Super</kbd>+<kbd>Space</kbd>, select "Diegeltheme" and everything should refresh automatically
